# Hanqi Jiang's Academic Homepage

这是一个学术个人主页网站，复刻自 https://hq0709.github.io/

## 文件结构

- `index.html` - 主页面文件
- `styles.css` - 样式文件
- `script.js` - JavaScript交互文件
- `README.md` - 说明文档

## 功能特性

- 响应式设计，支持移动端和桌面端
- 平滑滚动导航
- 现代化的UI设计
- 包含个人信息、教育背景、经历、发表论文等完整内容

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可查看网站
2. 或者使用本地服务器（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```
   然后在浏览器中访问 `http://localhost:8000`

## 自定义

你可以根据需要修改以下内容：
- 个人信息：编辑 `index.html` 中的相应部分
- 样式：修改 `styles.css` 中的CSS变量和样式
- 交互效果：修改 `script.js` 中的JavaScript代码

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
